TITLE  
Empty output after filtering OOV in Japanese TTS model

SYMPTOMS  
• The output text is empty after the line "text after filtering OOV."  
• No error messages are displayed during the execution of infer.py.  
• The expected output from the TTS model is not generated.

EXPECTED BEHAVIOR  
The software should produce a valid output text after filtering out out-of-vocabulary (OOV) words, allowing the TTS model to generate synthesized speech based on the provided input.

REPRODUCTION STEPS  
1. Download the Japanese TTS model using the command:  
   `wget https://dl.fbaipublicfiles.com/mms/tts/jvn.tar.gz`  
2. Run the inference script with the command:  
   `python infer.py`  
3. Observe the output after the line "text after filtering OOV."  
4. Note that the output is empty, indicating a potential issue with the OOV filtering process.

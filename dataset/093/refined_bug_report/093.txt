TITLE
Configuration conflict between `attn_num_mem_kv` and `attn_one_kv_head`

SYMPTOMS
• An error occurs when both `attn_num_mem_kv` and `attn_one_kv_head` are set in the `Decoder` configuration.
• The exact error message is not provided, but it suggests a conflict between these two parameters.

EXPECTED BEHAVIOR
• When setting `attn_num_mem_kv > 0` and `attn_one_kv_head = True`, the model should be able to process memory tokens while also using a single key-value head for attention without raising an error.
• The TransformerWrapper should initialize correctly and allow forward pass operations such as generating logits.

REPRODUCTION STEPS
1. Initialize the `TransformerWrapper` with specific parameters, including setting both `attn_num_mem_kv` and `attn_one_kv_head`.

TITLE  
MMS - Forced Alignment: Tensor Size Mismatch Error

SYMPTOMS  
• Encountered a runtime error when attempting to forced align an audio file with its text transcription.  
• Error message: `RuntimeError: Sizes of tensors must match except in dimension 1. Expected size 1649 but got size 1799 for tensor number 1 in the list.`  
• The process fails to complete the alignment and segmentation of the audio file.

EXPECTED BEHAVIOR  
The software should successfully align the audio file with its corresponding text transcription without any errors, producing the expected output files in the specified output directory.

REPRODUCTION STEPS  
1. Execute the command: `python align_and_segment.py --audio_filepath audio.wav --text_filepath text.txt --lang ful --outdir output --uroman uroman/bin`  
2. Ensure that the specified audio file (`audio.wav`) and text file (`text.txt`) are correctly formatted and accessible.  
3. Observe the console output for any error messages during the execution of the command.  
4. Confirm that the expected output files are generated in the specified output directory.

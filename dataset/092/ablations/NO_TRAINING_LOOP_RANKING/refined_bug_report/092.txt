TITLE  
Decoder outputs NaN logits with all-padding context in cross-attention

SYMPTOMS  
• Decoder logits output NaN values when the context provided to the cross-attention block consists entirely of padding.  
• The context mask is entirely set to `False`.  
• The output tensor `out` contains all NaN values.  

EXPECTED BEHAVIOR  
When the context is all padding, the decoder should handle this gracefully and produce valid logits, indicating that no cross-attention should be performed. Ideally, the output should reflect the absence of conditioning rather than resulting in NaN values.

REPRODUCTION STEPS  
1. Initialize the decoder using the `x-transformers` library version **1.26.0** with cross-attention enabled.  
2. Generate a random input tensor `i` with shape `(2, 20)`.  
3. Create a context tensor filled with random values with shape `(2, 4, 1024)`.  
4. Create a context mask tensor with shape `(2, 4)` filled with `False` values, indicating all padding.  
5. Call the decoder with the input tensor `i`, the context tensor, and the context mask.  
6. Observe that the output tensor `out` contains all NaN values.  

This structured analysis clarifies the problem, outlines the symptoms, sets expectations for correct behavior, and provides clear steps to reproduce the issue.

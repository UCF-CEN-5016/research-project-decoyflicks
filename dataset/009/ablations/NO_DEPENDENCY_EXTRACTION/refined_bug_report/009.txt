TITLE
AttributeError when exporting Faster R-CNN model

SYMPTOMS
• Encountered an `AttributeError` during the model export process.
• The specific error message received is: `AttributeError: 'DetectionFromImageModule' object has no attribute 'outputs'`.
• The issue arises after training the model using `model_main_tf2.py`.

EXPECTED BEHAVIOR
The model should export successfully without any errors, allowing for further usage in inference or deployment.

REPRODUCTION STEPS
1. Train the model using the command `model_main_tf2.py` with the configuration for `faster_rcnn_resnet50_v1_640x640_coco17_tpu-8`.
2. Attempt to export the trained model using `exporter_main_v2.py`.
3. Observe the error message during the export process. 

ENVIRONMENT
• CUDA version: 11.5
• TensorFlow GPU version: 2.4
• Python version: 3.7.16

TITLE
AttributeError due to missing 'outputs' attribute in 'DetectionFromImageModule'

SYMPTOMS
• Received `AttributeError: 'DetectionFromImageModule' object has no attribute 'outputs'` when attempting to export the model.
• The error occurred during the execution of `exporter_main_v2.py`.

EXPECTED BEHAVIOR
Upon successful execution of `exporter_main_v2.py`, the script should generate a TensorFlow 2.x SavedModel or frozen graph that can be used for inference.

REPRODUCTION STEPS
1. Ensure CUDA version is 11.5.
2. Install TensorFlow-GPU version 2.4 and Python version 3.7.16.
3. Train the `faster_rcnn_resnet50_v1_640x640_coco17_tpu-8` model using `model_main_tf2.py`.
4. Attempt to export the trained model with `exporter_main_v2.py`.
5. Observe the error: `AttributeError: 'DetectionFromImageModule' object has no attribute 'outputs'`.

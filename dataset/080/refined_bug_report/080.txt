TITLE
Validation Accuracy Exceeds Training Accuracy in ViT Model

SYMPTOMS
• Validation accuracy is consistently higher than training accuracy throughout the training process.
• This issue occurs both with and without dropout layers.

EXPECTED BEHAVIOR
For a typical machine learning model, including Vision Transformer (ViT), one would expect the training accuracy to be equal to or higher than the validation accuracy. The validation set should serve as an approximation of the test set performance, which is expected to be lower due to the model's generalization errors.

REPRODUCTION STEPS
1. Ensure you have the `vit_pytorch` library installed and the cats and dogs dataset downloaded.
2. Follow the example script provided in the documentation or repository for training the ViT model.
3. Set the parameters as specified:
- batch_size = 64
- model configuration:
```python
from vit_pytorch.vit_for_small_dataset import ViT
model = ViT(
image_size = 224,
patch_size = 16,
num_classes = 2,
dim = 1024,
depth = 6,
heads = 16,
mlp_dim = 2048,
dropout = 0.1,
emb_dropout = 0.1)
```
4. Train the model using the training and validation datasets.
5. Plot the training and validation accuracy to observe the discrepancy.

Note: The plot provided in the bug report shows `SmallDatasetVit` compared to a CNN for reference, which can be helpful but is not strictly necessary for reproducing the issue.

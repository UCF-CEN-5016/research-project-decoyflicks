TITLE  
Unexpected keyword argument 'token' in ParquetConfig during script execution

SYMPTOMS  
• The script `test_tune.sh` fails to execute properly.  
• The following error message is displayed:  
  `ParquetConfig.__init__() got an unexpected keyword argument 'token'`  
• The script attempts to use DeepSpeed but fails to find an optimal configuration, leading to the message:  
  `No optimal DeepSpeed configuration found by autotuning.`  
• There are multiple permission denied errors related to SSH access:  
  `localhost: Permission denied, please try again.`  
  `localhost: daimyollc@localhost: Permission denied (publickey,password).`  

EXPECTED BEHAVIOR  
The script should execute without errors, successfully running the autotuning process for the model with DeepSpeed, and finding an optimal configuration.

REPRODUCTION STEPS  
1. Clone the Hugging Face Transformers repository:  
   ```bash
   git clone https://github.com/huggingface/transformers.git
   ```
2. Navigate to the cloned directory:  
   ```bash
   cd transformers
   ```
3. Install the required package:  
   ```bash
   pip install .
   ```
4. Create a new directory for testing and navigate into it:  
   ```bash
   mkdir ../test && cd ../test
   ```
5. Install DeepSpeed and any other dependencies:  
   ```bash
   pip install deepspeed
   ```
6. Create a new script file and make it executable:  
   ```bash
   touch test_tune.sh
   chmod +x test_tune.sh
   ```
7. Copy the content of `test_tune.sh` from [`autotuning/hf/gpt2-xl/test_tune.sh`](https://github.com/microsoft/DeepSpeedExamples/blob/ff9a0234cf22dd9af03c5c7aa8037fb9143adca6/training/autotuning/hf/gpt2-xl/test_tune.sh) into the newly created file.
8. Modify the script variables to set `HF_PATH=../` and `NGPUS=6`.
9. Execute the script with the tune command:  
   ```bash
   ./test_tune.sh tune
   ```

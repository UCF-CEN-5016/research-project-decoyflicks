**TITLE**  
Compatibility issue between tf-models-official==2.12.0 and Python 3.8 causing TypeError during training evaluation.

**SYMPTOMS**  
• Encountered a `TypeError` when attempting to run the training and evaluation of the model.  
• Error message:  
  ```

          return passthrough_logs | logs
      TypeError: unsupported operand type(s) for |: 'dict' and 'dict'
  ```  
• The issue arises specifically during the execution of the `run_experiment` function.

When the software functions correctly, the model should train and evaluate without any errors, returning the training and evaluation logs as expected.

TITLE  
ImportError when importing tensorflow_models due to undefined symbol in tensorflow_text

SYMPTOMS  
• ImportError occurs when trying to import `tensorflow_models` as `tfm`.  
• The following error message is displayed:  
  ```
  ImportError: /usr/local/lib/python3.10/dist-packages/tensorflow_text/core/pybinds/tflite_registrar.so: undefined symbol: _ZN4absl12lts_2022062320raw_logging_internal21internal_log_functionB5cxx11E
  ```  
• The import fails, preventing the use of TensorFlow models.

EXPECTED BEHAVIOR  
When the software works correctly, importing `tensorflow_models` should succeed without any errors, allowing access to the functionalities provided by the library.

REPRODUCTION STEPS  
1. Ensure you are using the latest version of TensorFlow Model Garden and TensorFlow 2.  
2. Attempt to import the library using the command:  
   ```python
   import tensorflow_models as tfm
   ```  
3. Observe the output for any errors.

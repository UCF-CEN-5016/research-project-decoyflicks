TITLE

  ```

If you're using `model.compile()`, did you forget to provide a `loss` argument?

• The issue began occurring after the code was functioning correctly for a month.

EXPECTED BEHAVIOR  
The model should train without warnings, and gradients should be computed for all variables involved in minimizing the loss.

REPRODUCTION STEPS

2. Prepare a custom dataset for training.  
3. Execute the training script using "model_main_tf2.py".

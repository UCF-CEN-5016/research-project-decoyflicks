EXPECTED BEHAVIOR  
The test script should execute without errors, producing a sequence of translated sentences and a calculated BLEU score.

1. Set up the environment with Keras 2.15, TensorFlow, and the required dependencies on Linux Parrot OS 5.3.  
2. Use Python 3.9 and ensure the GPU (GeForce RTX 3070 Mobile) is correctly configured.  
3. Run the provided test script for neural machine translation, ensuring to include the BLEU score evaluation from NLTK.  
4. Observe the segmentation fault occurring during the execution of the `GreedySampler()`.  
5. Check the output logs for error messages related to tensor shapes and segmentation faults.

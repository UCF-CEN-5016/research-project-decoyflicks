TITLE  
NaN Loss During Training with FP8 Precision in Transformer Engine

SYMPTOMS  
• Loss value becomes NaN during training when using FP8 precision.  
• No specific error messages are displayed; the training process simply reports NaN for the loss.  
• The training process halts or becomes ineffective due to the NaN loss values.  

EXPECTED BEHAVIOR  
The training process should proceed without encountering NaN loss values, resulting in a valid numerical loss that decreases over iterations.

REPRODUCTION STEPS  
1. Set up a transformer model using the Transformer Engine library.  
2. Configure the model to use FP8 precision for training.  
3. Initiate the training process with a dataset.  
4. Monitor the loss value during training to observe if it becomes NaN.

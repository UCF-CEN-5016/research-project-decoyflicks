• The issue seems to have emerged after a specific commit in the repository.  

EXPECTED BEHAVIOR  
The software should successfully perform forced alignment on the provided audio file and text transcription, producing the expected output files without any shape-related errors.

REPRODUCTION STEPS  
1. Run the command: `python align_and_segment.py --audio_filepath audio.wav --text_filepath text.txt --lang udm --outdir output --uroman uroman/bin`.

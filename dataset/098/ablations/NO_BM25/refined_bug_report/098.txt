TITLE  
NameError due to unassigned variable in SinusoidalPosEmb class

SYMPTOMS  
• The `theta` parameter is passed to the `__init__` method but not assigned as a member variable.  
• When executing the `forward` function, a `NameError` is raised because `theta` is not defined within the method scope.  
• The error message displayed is:  
```
NameError: name 'theta' is not defined
```

EXPECTED BEHAVIOR  
The `SinusoidalPosEmb` class should correctly use the `theta` parameter in the `forward` method without raising an error. The output should be a tensor representing the sinusoidal positional embeddings based on the input tensor `x`.

REPRODUCTION STEPS  
1. Create an instance of the `SinusoidalPosEmb` class with a specified dimension (e.g., `dim = 4`).  
2. Prepare an input tensor `x` (e.g., `torch.ones((10))`).  
3. Call the `forward` method of the `SinusoidalPosEmb` instance with the input tensor `x`.  
4. Observe the raised `NameError`.

TITLE  
ModuleNotFoundError: Unable to locate 'commons' module during inference

SYMPTOMS  
• Encountering a `ModuleNotFoundError` when executing `infer.py`  
• Error message: `ModuleNotFoundError: No module named 'commons'`  
• Inability to generate the expected output (audio file) from the provided input  

EXPECTED BEHAVIOR  
The program should successfully execute the `infer.py` script, generating the expected audio output file based on the input text and audio parameters without any module-related errors.  

REPRODUCTION STEPS  
1. Clone the fairseq repository.  
2. Install the package in editable mode using `pip install --editable ./`.  
3. Change the directory to `examples/mms/tts/`.  
4. Execute the command: `python infer.py --model-dir model/ --wav test.wav --txt "Heute ist ein schöner Tag."`.
